# Архитектура
## Блоки
- Mobile App - регистрирует устройство, получает пуши.
- Device Registry - хранит токены устройств.
- Бизнес-сервисы - генерируют события.
- Event Bus – очередь сообщений для событий, подписка Notification Orchestrator.
- Notification Orchestrator - фильтрует события по пользователям, шаблонам, предпочтениям; создаёт задачи на пуши.
- Push Workers - отправляют пуши на платформы FCM/APNs, обрабатывают ошибки.
- FCM – отправка пушей на Android-устройства.
- APNs – отправка пушей на iOS-устройства.
- Мониторинг – обновляет токены, собирает статистику доставки и открытий, метрики для аналитики.
