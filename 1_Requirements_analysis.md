## 2. Исправленная версия ТЗ
1. Пользователь может добавить в корзину от 1 до 10 единиц одного товара.
2. Пользователь может изменить количество каждого товара в корзине от 1 до 10 единиц.
3. Для удаления товара из корзины используется кнопка "2Удалить" или уменьшение количества до 0.
4. При удалении товара из корзины появляется окно для подтверждения удаления товара из корзины.
5. Если пользователь не подтверждает удаление, товар остается в корзине в предыдущем количестве единиц.  
6. После подтверждения удаления товар исчезает из корзины, а суммарная стоимость пересчитывается.
7. В корзине может находиться не более 5 различных товаров.
8. Суммарное количество всех товаров в корзине не может превышать 50 штук.
9. При попытке добавить товар, превышающий лимит по количеству единиц одного товара, система показывает сообщение: "Лимит товара превышен. Вы не можете добавить больше 10 единиц одного товара".
10. При попытке добавить товар, превышающий лимит по количеству разных товаров, система показывает сообщение: "В корзине может быть только 5 видов разных товаров".
11. Цена на товар фиксируется на момент добавления в корзину.
12. На странице корзины отображается список товаров, их количество, цена за единицу и общая стоимость позиции. Также отображается итоговая сумма всех товаров в корзине. Валюта должна быть явно указана.
13. В корзине может быть реклама других продуктов.
14. Реклама товаров должна показываться с понедельника по пятницу включительно во временные промежутки 8:00-12:00 и 19:00-22:00. Формат рекламы должен быть определен заранее.
15. Если пользователь удалит все товары из корзины, система показывает сообщение "Корзина пуста".
    
## 3. Уточняющие вопросы 
1. Какое суммарное количество всех товаров действительно может быть в корзине?
2. Нужно ли показывать разные сообщения о превышении лимитов (по количеству одного товара, по количеству разных товаров, по суммарному количеству) или достаточно одного общего сообщения?
3. Какой формат рекламы предполагается для отображения в корзине?
4. Будет ли система автоматически обновлять цену на товары в корзине при изменении цены в каталоге или цена фиксируется на момент добавления?
5. Нужно ли показывать пользователю, сколько единиц товара ещё можно добавить перед превышением лимита?
6. Должна ли корзина сохраняться между сессиями пользователя или всегда начинается пустой?
